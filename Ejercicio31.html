Introducción

SQL (Structured Query Language) es un lenguaje de programación utilizado para gestionar bases de datos relacionales. Permite realizar operaciones como la creación, modificación y consulta de datos en una base de datos.

show database

El comando SHOW DATABASES se utiliza para mostrar todas las bases de datos disponibles en el servidor MySQL.

Creación de una tabla y mostrar sus campos

Para crear una tabla en MySQL, se utiliza el comando CREATE TABLE, seguido de la definición de los campos y sus tipos de datos.

Carga de registros a una tabla y su recuperación

Usamos "INSERT INTO" nombre_tabla (campo1, campo2) VALUES (valor1, valor2)" para insertar datos en la tabla.
